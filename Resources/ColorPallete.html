Color in Html Explained
Integer numbers in html can be represented as
Decimal numbers (dec), 0 to 9
Hexadecimal (hex) 0 to 9 then a to f (representing 10 to 15)
example: hex of 10 is 16+0 in dec

A byte is the basic numerical unit in computing
a byte has a value of 0 to 255 in dec, and #00 to #ff in hex (# means what follows is a hex and not decimal number)

Color in html is represented by 3 bytes, r for red, g for green, and b for blue
In html color is represented by 3 numbers, each 0-255. These are red, blue, green and transparency
The format is either rgb(r,g,b) or in hex code #rrggbb

More recently, a 4th number (alpha) for transparency can be added
for decimal it is a number between 0 and 1. 0 is transparent, 1 is solid
for hex it is 00 to ff, 00 is transparent and ff is solid
for example
rgb(255,0,0,1) rgb(255,0,0,0.5)
#ff0000ff; #ff000066;

An alternative hex format 3 (rgb) or 4(rgba) numbers where each is between 0 (minimum) and f (maximum)

For the rest of this page, all color code used are in the rgb format.
The programmer can choose to add alpha when it is appropriate


Color Pallete
A color pallete is a set of complementary colors
This means that the colors in the set look good when they are shown next to each other
Usually a pallete consists of 5 colors

When designing a web page therefore,
the colors used (other than in imported images) should be from the same pallete

This page presents some plain colors and some pallete that I have collected
