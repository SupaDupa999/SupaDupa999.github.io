Lesson 0: How to work with SupaDupa

This lesson will take Harvey through all the procedures in navigating SupaDupa project

For each exercise, the supervisor (Dad or Grandpa) will

  • Guide Harvey through the procedures
  • Ask Harvey to repeat the procedures until he can perform all of them without help

Exercise 1: Handling his email account

  1. From the computer, activate the email client
  2. Navigate to the in box
  3. Identify the latest email received
  4. Identify an email from github
  5. Read the mrssage and understand what the validation number is all ablut
  6. Exit the email client

Exercise 2: Navigate the computer files Using the File Explorer

  1. From the computer, the File Explorer
  2. Navigate to folder SipaDupaFiles
  3. Navigate to all subfolders
  4. Identify the function of each folder
  5. Find a named file, starting at disk, before any folder

Exercise 3: Navigate the Home page of SupaDupa999
  1. Activate the browser
  2. Reach SupaDupa home page
  3. Find the following sets of information
    • Log in and out
    • Help with repositories
    • Help with files
    • Description of project map

Exercise 4: Sign in
  1. Switch on the email client
  2. Go to SupaDupa999 home page
  3. access encrypted password pdf file
    • Enter password
    • Read and understand how to use the passwords
  4. From the home page, sign in
    • Link to SupaDupa site
    • Identify and click the sign in button
    • Enter email address and account password
    • If confronted with demands for validation number
      • Go to email client
      • Idertify the email received
      • Read the validation number
      • Back to the sign in
      • Enter the validation number
    • Sign in

Exercise 4: Navigate SupaDupa999 From the sign in main page
  1. Visit in turn all repositories and sub-repositories
  2. Explain the intended use of each repository
  3. Identify each file and explain their functions
End of Lesson 0

If Harvey can repeat all these procedures without help, then he is ready to build a website
