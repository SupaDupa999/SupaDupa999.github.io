Lesson 1: Create a Webpage

Educational Objectives

At the end of this lesson, Harvey should achieve the following
  1. Able to create a new web page by 3 different methods
    • By copying another web page accessible from the www
    • By uploading a template file stored in his computer
    • By creating a blank file and coding the file
  2. To be able to conduct the following procedures
    • To examine both front and back s of a web page in the browser
    • To be able to upload a file from the computer to SupaDupa
    • To be able to edit and rename a file in SupaDupa
    • To be able to create multiple views (tabs) on the browser and switch from one to the other
    • To be able to mark, copy, paste, and delete blocks of text during editing
  3. To understand
    • The structure of a html file
    • The functions of the basic tags, html, head, body, meta, and title

Exercise 1: Creating a web page by copying another one

Step 1. Get a web page and get to its back end
  1. On the browser, create a new tab, and get http://SupaDupa1802.github.io/Cources/T_1.html
    a web page on Grandpa' SupaDupa site
  2. After loading grandpa's page, get to the back end of the page
    • At the browser's menu bar
    • Click Tools->Browsing Tools->Page Source
    • You should see the back end in a new tab
Step 2. Start a new blank page
  1. Open a new tab on the browser, and sign into SupaDupa999
  2. Navigate to the repository SupaDupa999.github.io
  3. Half way to the right and to the top of io, find a little button with + on it
  4. Click the button and select Create new file
  5. An empty edit box and an empty name box will be shown
Step 3. Copy content from source back end to clipboard
  1. Go to the tab with the back end of grandpa's page
  2. Copy the whole back end to the clipboard
    Mark out the whole content, then
    copy by pressing the ctrl and c button at the same time
Step 4. Paste content of clipboard to the editing box in SupaDupa999
  1. Go to the tab where the edit box of SupaDupa is
  2. Paste the content of the clipboard to the edit box
    use the mouse, move the pointer to the text box, and click left once
    paste by pressing ctrl and v keys at the same time
  3. Check that the content from grandpa's page is now in the text box
Step 5. Edit and save the file on SupaDupa
  1. Change the text between the body tags
  2. Replace Lesson 1: Create a Webpage between the 2 h2 tags with Success 1: Webpage S_1 Created
  3. Replace T_1 between the 2 title tags with S_1
  4. Enter a name S_1.html in the name box Above the edit box
  5. Click the Commit Changes button on the right of the edit box
Step 6. Check results
  1. Start a new tab on the browser, in it
  2. Get the new web page http://SupaDupa999.github.io/S_1.html
  3. Compare T_1.html from grandpa and S_1.html Harvey just created, and see they are the same

Exercise 3: Creating a web page by direct coding

The purpose of this exercise is to understand the basic structure of a html page

Step 1. Initiate a black page in an edit box

  1. Log in, navigate to the io repository
  2. Click the + button and create a black page in the editing box
Step 2. Insert the <html> and </html> tags
  1. Type the two tags in, separated by a few lines
  2. Explain the a web page begins and ends at the two tags
  3. Explain that the contents of a page is between the two tags
Step 3. Insert the <head> and </head> tags
  1. Type the two tags in, separated by a few lines
  2. Explain the the header starts and ends with these 2 tags
  3. Explain that the contents of the header is between the two tags
  4. Explain that the contents of the header is for
    • Telling the browser how the page should be put together and what it should look loke
    • To accommodate computer programs (Javascript)
Step 4. Insert the <meta name="viewport" content="width=device-width, initial-scale=1">
  1. Copy the line and paste it immediately after the <head> tag
  2. Explain that this makes sure that the page will always fit into the view port
    so that it can be properly seen in computer, Ipads, and Phones
    also explain that most coders just copy and paste this so no need to remember the details
Step 5. Insert the <title> and </title> tags
  1. Type the 2 tages, including a short name between them, after the meta tags
  2. Explain that whats between the title tags will show in the little sign
    on top of the main view of the browser
Step 6. Insert the <body> and </body> tags
  1. Type the two tags in, separated by a few lines
  2. Explains the body begins and ends between these tage
  3. Explain that the contents of the body are between these tags
  4. Explain that the contents are what the coder (Harvey) wants everyone to see
Step 7. Insert some text into the body

Step 8. Save the file

  1. Enter the name S_3.html into the name box above the editing box
  2. Click the Commit Changes button
Step 9. Compare the 3 pages created in this lesson
  1. At the browser, load the 3 files, each in its own tab
  2. Navigate between them and compare them
